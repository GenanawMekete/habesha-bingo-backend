# Telegram BINGO Game Backend

## Features
- User authentication via Telegram
- Card management and generation
- Game session handling
- Payment processing (Stripe, Telegram Stars)
- Real-time game updates (via WebSockets)
- Admin dashboard

## Prerequisites
- Node.js 18+
- MongoDB 6+
- Redis 7+ (for caching and sessions)
- Telegram Bot Token
